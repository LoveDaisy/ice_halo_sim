# GUI 使用指南

本文档介绍 Lumice GUI 应用 — 一个用于交互式配置和预览冰晕模拟的图形界面。

## 构建和运行

```bash
# 构建 GUI 应用
./build.sh -gj release

# 运行
./build/cmake_install/LumiceGUI
```

GUI 需要 display server 和支持 OpenGL 3.2 Core Profile 的 GPU。

## 界面布局

界面分为以下区域：

- **顶栏**：文件操作（New、Open、Save、Save As）和模拟控制（Run、Stop、Revert）
- **左侧面板**：配置标签页（Crystal、Scene、Render、Filter）
- **晶体预览**：当前晶体的 3D 线框/着色预览（在 Crystal 标签页中）
- **渲染预览**：镜头投影的冰晕模拟结果（在 Render 标签页中）
- **浮动镜头栏**：快速镜头类型选择器（Render 标签页激活时可见）
- **状态栏**：当前文件路径、模拟状态和修改指示

## 配置标签页

### Crystal 标签页

配置冰晶几何形状和朝向：

- **晶体类型**：`Prism`（六棱柱）或 `Pyramid`（六棱锥）
- **形状参数**：高度比、面距离，以及六棱锥特有参数（上/下高度、Miller 指数）
- **轴分布**：天顶角、自转角和方位角 — 可设为固定值、均匀分布或高斯分布
- **多晶体**：支持增删晶体，删除时有确认弹窗
- **3D 预览**：交互式 trackball 旋转和缩放；支持 4 种渲染风格：
  - **Wireframe**（线框）：仅显示边
  - **Hidden Line**（隐藏线）：带隐藏线消除的边（默认）
  - **X-Ray**（透视）：透明显示内部结构
  - **Shaded**（着色）：基于法线的实体着色

### Scene 标签页

配置模拟参数：

- **太阳位置**：高度角和方位角（度）
- **光谱**：标准光源（如 D65）或自定义波长-权重对
- **光线数**：追踪光线数量
- **最大碰撞次数**：每条光线的最大晶体交互次数
- **散射层**：定义多次散射场景，包括晶体比例和可选过滤器

### Render 标签页

配置渲染输出：

- **镜头类型**：支持 8 种 — Linear、Fisheye Equal Area、Fisheye Equidistant、Fisheye Stereographic、Dual Fisheye（3 种变体）、Rectangular
- **视场角**：可调节 FOV（度）
- **视角**：俯仰角、方位角和横滚角
- **分辨率**：输出图像尺寸
- **颜色**：背景色、光线颜色和不透明度
- **多渲染器**：支持增删渲染配置

### Filter 标签页

配置光线路径过滤器：

- **光路过滤器**：指定面序列，支持对称选项（P/B/D）
- **入射/出射过滤器**：按入射面和出射面过滤
- **多过滤器**：支持增删过滤器配置

## 文件操作

### 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| Ctrl+N | 新建项目 |
| Ctrl+O | 打开项目 |
| Ctrl+S | 保存项目 |
| Ctrl+Shift+S | 另存为 |
| Ctrl+Enter | 运行模拟 |
| Ctrl+K | 停止模拟 |

### 项目文件格式（`.lmc`）

Lumice 使用二进制项目文件格式（`.lmc`），存储：

- **配置**：所有晶体、场景、渲染和过滤器设置，以语义化 JSON 形式保存
- **预览纹理**：可选的内嵌 PNG，保存最近一次渲染结果

格式使用 44 字节头部，包含魔数 `LMC\0`、版本字段和 JSON/纹理的偏移量/大小指针。值以人类可读的语义类型存储（如 `"prism"` 而非枚举索引），确保前向兼容性。

### 未保存的更改

当存在未保存的更改时（状态栏中有修改指示），应用会在以下操作前警告：
- 新建项目
- 打开另一个项目
- 关闭应用

## 模拟工作流

1. **配置**：在左侧面板设置晶体、场景、渲染和过滤器
2. **运行**（Ctrl+Enter）：当前配置被序列化并提交到模拟核心
3. **监控**：状态栏显示模拟进度；模拟期间参数被禁用
4. **查看**：结果可用时显示在渲染预览中
5. **停止**（Ctrl+K）：需要时可提前终止模拟
6. **恢复**：恢复到最近一次提交的配置（撤销运行后所做的参数修改）

### 模拟状态

- **Idle**（空闲）：无模拟运行；所有参数可编辑
- **Simulating**（模拟中）：模拟进行中；参数被禁用
- **Done**（完成）：模拟完成；预览中有结果
- **Modified**（已修改）：模拟完成后修改了参数
