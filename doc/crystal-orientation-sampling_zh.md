# 晶体取向采样

本文档描述 Lumice 如何在蒙特卡洛（Monte Carlo）冰晕模拟中对晶体 c 轴取向进行采样。

## 1. 物理模型

大气中的冰晶在气动力矩（aerodynamic torque）与随机扰动（stochastic perturbation）的共同作用下确定取向。单个六棱柱晶体的取向由三个角度完全描述：

- **天顶角（zenith angle, θ）**：晶体 c 轴与竖直方向之间的夹角，是决定晕形几何形态的主要因素。
- **方位角（azimuth angle, φ）**：c 轴投影绕竖直方向的旋转角。对于自由下落的晶体，此角服从均匀分布。
- **自转角（roll angle, ψ）**：晶体绕自身 c 轴的旋转角。

这种分解具有物理动机：重力定义了一个优选方向（竖直方向），而气动阻力主要沿此方向作用。由此产生的取向球面上的概率分布是一个**纬度带（zonal band）**——集中在某一优选余纬度附近，方位角均匀分布。这与旋转对称的球冠（cap）有本质区别：对称轴是重力矢量，而非晶体的 c 轴。这正是 von Mises–Fisher（vMF）分布不适用的原因——vMF 产生的是球冠而非纬度带。


## 2. 冰晶运动模式与分布选择

天顶角分布取决于雷诺数（Reynolds number, Re），而雷诺数由晶体尺寸、形状和下落速度决定。存在三种不同的运动状态，每种需要不同的统计模型。

### 2.1 稳定模式（低 Re）

在低雷诺数下，气动力矩将晶体恢复至平衡取向（板状晶体的 c 轴趋向竖直，柱状晶体的 c 轴趋向水平）。布朗运动和大气湍流引入小角度涨落。所得倾斜角服从**高斯分布（Gaussian distribution）**：

```
p(θ) ∝ exp(-(θ - μ)² / 2σ²)
```

其中 σ 通常为 0.4°–3°。偏振激光雷达反演结果表明，良好取向的板状晶体 σ ≈ 1°–2°（Noel & Sassen 2005），卫星观测也证实约 80% 的取向晶体偏离水平方向不超过 3°（Noel & Chepfer 2004）。

这是产生所有可辨认晕现象的运动状态：幻日（sundogs）、光柱（light pillars）、环天顶弧（circumzenith arcs）和幻日环（parhelic circles）。一个关键认识涉及**物理自洽性（physical self-consistency）**：产生可辨认晕的晶体*必须*具有高质量的取向（σ ≤ 数度），这恰好对应于高斯分布精确成立的稳定模式。锐利的晕需要窄的倾斜分布，而窄的倾斜分布仅在恢复力矩占主导时才会出现。这并非巧合，而是同一物理过程同时控制运动和光学的必然结果。

### 2.2 不稳定模式（高 Re）

当雷诺数超过临界值（六角板状晶体 Re > ~237）时，尾流变得不稳定，晶体进入振荡或混沌运动。

**之字形振荡（zigzag oscillation）。** 晶体来回摆动，倾斜角为：

```
θ(t) = |A · sin(ωt) + B|
```

其中 A 为振荡幅度，B 为平均倾斜偏移量。时间平均分布为**折叠反正弦分布（folded arcsine）**——呈 U 型，峰值出现在转折点附近。Stout 等人（2024）通过实验证实，高斯模型在这些不稳定模式下从定性上就是失败的。

**螺旋运动（spiral motion）。** 晶体绕竖直方向进动，倾斜角相对稳定。可以用均值偏移的高斯分布（μ ≠ 0）近似。

这两种模式产生的是弥散的背景光，而非锐利的晕。柱状晶体可能表现出特征性的倾斜角；Zhong 等人（2023）报告柱状晶体倾斜模式的 θ_e ≈ 38°。

### 2.3 尺寸聚合倾斜（size-aggregated tilt）

真实冰云中的晶体跨越一系列尺寸。对每种尺寸，σ 由 Re 决定；晶体越大，σ 越大。当倾斜角在完整尺寸分布上积分后，结果服从**拉普拉斯分布（Laplace / double-exponential distribution）**：

```
p(θ) ∝ exp(-|θ - μ| / b)
```

推导过程：若 X|σ ~ N(0, σ²) 且 σ² ~ Exp(λ)，则 X ~ Laplace(0, b)，其中 b = 1/√(2λ)。小晶体贡献锐利的高斯核心，大晶体贡献更重的指数尾部。

深空闪光观测证实了这一预测。Kostinski、Marshak 与 Várnai（2025）发现角衰减遵循 exp(−δ/s)，这恰好是拉普拉斯特征，数据来自 DSCOVR 航天器。

拉普拉斯分布简化了配置：无需使用多个具有不同高斯 σ 值的晶体条目，只需一个具有拉普拉斯尺度参数的条目即可。


## 3. 支持的分布类型

支持四种分布类型以及一种确定性模式。所有类型在 JSON 中共享相同的 `mean`/`std` 参数字段；`std` 的含义因类型而异。

| 类型 | 定义 | `mean` | `std` | 适用场景 |
|------|------|--------|-------|----------|
| `gauss` | p(θ) ∝ exp(−(θ−μ)²/2σ²) | 中心值 μ（度） | 标准差 σ（度） | 稳定模式取向 |
| `uniform` | p(θ) = 常数，定义域 [μ−w/2, μ+w/2] | 中心值 μ（度） | 全宽 w（度） | 随机取向；自转角 |
| `zigzag` | θ = \|A·sin(2πU) + B\|，折叠反正弦 | 倾斜偏移 B（度） | 幅度 A（度） | 大晶体振荡 |
| `laplacian` | p(θ) ∝ exp(−\|θ−μ\|/b) | 中心值 μ（度） | 尺度参数 b（度） | 尺寸聚合倾斜 |

JSON 中的**标量值**（如 `"zenith": 90`）视为确定性模式：每个晶体均接收该精确角度，不引入随机化。

**参数说明：**
- `gauss` σ = 2° 意味着 68% 的晶体落在均值 ±2° 范围内。
- `uniform` 的 `std` 是全范围宽度，而非标准差。
- `zigzag` 分布在折叠后的支撑集（support）为 [0, |A|+|B|]。
- `laplacian` 的标准差为 √2·b ≈ 1.41b。


## 4. 采样实现

### 4.1 雅可比校正（Jacobian correction）

在球面上采样角度 θ 需要考虑球面面积元素 sin(θ)。目标密度为：

```
p_sphere(θ) ∝ p_distribution(θ) · sin(θ)
```

若不做此校正，采样会过度集中在 sin(θ) → 0 的极点附近。

### 4.2 采样路径

系统自动选择三条路径之一：

**路径 1：瑞利分布（Rayleigh）（余纬度 < 0.5°）。** 在极点附近，雅可比因子与高斯分布结合后在切平面上给出瑞利分布。采样器抽取二维高斯样本，无需拒绝即可产生正确的球面密度。精确，每个样本 O(1)。

**路径 2：通用拒绝采样（generic rejection）。** 适用于高斯（非瑞利情形）、之字形和拉普拉斯分布：

1. 从基础分布中抽取候选 θ。
2. 以概率 cos(φ)/M 接受，其中 φ 为经 `NormalizeLatitude` 折叠后的纬度，M 为包络常数。

`ComputeJacobianEnvelope()` 计算的包络常数：

| 类型 | 包络常数 M | 覆盖范围 |
|------|-----------|----------|
| 高斯（Gaussian） | cos(max(\|μ\| − 3σ, 0)°) | 99.7% |
| 之字形（Zigzag） | cos(max(\|μ\| − A, 0)°) | 全幅度 |
| 拉普拉斯（Laplacian） | cos(max(\|μ\| − 5b, 0)°) | 99.3% |

**路径 3：确定性（deterministic）。** 无需采样。

### 4.3 各类型的候选生成

- **高斯：** 正态变量 × σ + μ。每个候选 O(1)。
- **之字形：** θ = |A · sin(2πU) + B|，U ~ Uniform(0,1)。O(1)。
- **拉普拉斯：** 逆 CDF 方法：θ = μ − b · sign(U−0.5) · ln(1−2|U−0.5|)。O(1)。
- **均匀：** 从 [μ − w/2, μ + w/2] 中抽取。O(1)。

所有候选均经过雅可比拒绝步骤（路径 2）。

### 4.4 接受率

- 赤道附近（天顶角 ≈ 90°）：~100%（cos φ ≈ 1）
- 中纬度（天顶角 ≈ 45°）：~70–80%
- 极点附近（天顶角 ≈ 5°）：~30–40%
- 之字形/拉普拉斯分布：接受率类似，由相同的 cos(φ)/M 公式决定


## 5. 历史背景

原始实现直接从分布中采样天顶角，未做 sin(θ) 雅可比校正。误差大小取决于平均天顶角：

| 平均天顶角 | 畸变倍率 | 受影响的类型 |
|-----------|---------|-------------|
| 90° | 1.0× | 柱状晶体晕 |
| 45° | 1.4× | — |
| 10° | 5.8× | 接近竖直的板状晶体晕 |
| 0° | ∞ | 完美取向的板状晶体 |

该修复在 explore-zenith-sampling（2026-04）期间实现，同时评估并否决了 vMF、Matrix Fisher 和 Bingham 分布。
